# PCB Play - PCB Design Learning Platform

A comprehensive platform for learning PCB design with interactive tools, component library, and AI-powered error detection.

## Features

### 🎯 Backend Features
- **Authentication System**: Register/Login with JWT
- **Components Database**: 200+ PCB/IoT components with specifications
- **Project Management**: Create/Read/Update/Delete projects
- **KiCad Integration**: File upload/download and processing
- **Error Detection System**: AI-powered error checking
- **RESTful API**: Proper error handling and validation
- **MySQL Database**: Proper relations and indexing

### 🎨 Frontend Features
- **Modern UI/UX**: Responsive design with Material-UI
- **Component Learning Platform**: Search, filter, and learn about components
- **PCB Design Workspace**: KiCad integration and project management
- **Error Checker Tool**: Visualization and solutions
- **User Authentication**: Complete auth flows
- **State Management**: Redux Toolkit
- **Type Safety**: TypeScript
- **Real-time Updates**: Socket.io integration

### 📚 Learning Resources
- 200+ components with detailed specifications
- Pin configurations and footprints
- Common mistakes and solutions
- Design best practices
- KiCad tutorials
- PCB design guidelines

## Tech Stack

### Backend
- Node.js with Express
- MySQL database
- JWT authentication
- Multer for file uploads
- Natural language processing for error detection
- Socket.io for real-time updates

### Frontend
- React 18 with TypeScript
- Redux Toolkit for state management
- Material-UI for components
- React Router for navigation
- React Query for data fetching
- Recharts for data visualization
